HTML

HTML stands for Hyper Text Markup Language.

It is used to design web pages using markups.

It uses tags to sturcture content.

Elements are building blocks of HTML.

HTML documents are saved with .html extension.

They can be opened in any browser.

HTML5 is the latest version.

It introduced new semantic elements.

Example elements: 'article','section', 'nav'

HTML is not a programming language.

<!DOCTYPE html> <html></html> <head></head> <body></body> <p>Paragraph</p>

CSS

CSS stands for Cascading Style Sheets.

CSS is a style sheet language used for formatting content in HTML webpages.

CSS style sheets can define the appearance and formatting of text, tables, and other elements separately from the content itself.

You can apply color, fonts, spacing, etc.

CSS can be responsive using media queries.

Latest version of CSS is CSS3.

CSS is essential for front-end design.

It uses properties and values.

CSS uses selectors to target HTML elements.

CSS helps web developers create a uniform look across an entire website.

color: blue; font-size: 16px; margin: 0 auto; display: flex; text-align: center;
  • Selectors
  • Properties
  • Values
  • Box Model
  • Media Queries

Git and Github

Git is a version control system.

It helps track code changes.

Github is a platform to host repositories.

Supports collaboration on code.

You can clone, commit, push, pull code.

Braches allow feature development.

Merge combines changes.

Pull request review code before merging.

Git is used via command line or GUI.

Git is popular for open source project.

git clone url git status git add git push origin main git commit -m "message"

Responsive Design

Responsive design adapts to screen size.

Uses flexible layout media queries.

Important for mobile friendly sites.

Viewport helps control layout scaling.

Responsive images scales with screens.

CSS flexbox and grid helps layout.

Rem and em units are scalable.

Always test on multipl devices.

Mobile-first design is recommended.

Frameworks like bootstrap helps too.

width: 100%; max-width: 100%; height: auto; display: grid; @media (max-width: 600px) {};

  • Flexbox
  • Grid
  • Moblie-First
  • Media Queries
